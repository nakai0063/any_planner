## 2. 画面構成の詳細

### 2.1 タイムラインビュー（メイン画面）

- 縦方向の時間軸。起床〜就寝の範囲で表示
- タスクが時間軸上にカード形式で配置
- **空き時間が自動で数値表示**（「あと何時間使えるか」が見える）
- カレンダー連携した予定を同一タイムライン上に統合表示
- 時間軸が左側、タスクカードが右側

### 2.2 あとでやるリスト（Later List）

- 時間未定タスクの一時保管領域
- タイムライン画面の下部/ボトムシートからアクセス
- ドラッグでタイムラインへ移動可能
- 「とにかく突っ込む」運用を明確に推奨

### 2.3 タスク登録画面

- タスク種別3択: **あとで / 時間指定 / 繰り返し**
- 開始時間ピッカー（v1.0.17で改善）
- ノート欄（長文スクロール対応）
- カスタムアイコン設定（v1.0.17〜、タスク名から自動サジェスト）
- タスクコピー機能（v1.0.17〜）

### 2.4 統計画面

- 完了タスク数
- 連続達成日数（ストリーク）
- 時間帯別の生産性グラフ
- PRO版で週間・月間統計

### 2.5 設定画面

- 起床/就寝時間設定
- カレンダー連携管理
- テーマ切替
- 通知設定

### 2.6 ウィジェット

- ホーム画面ウィジェット対応
- 次のタスク、あとでやるリストを表示
- Androidではスクロール可能ウィジェット
- 表示カレンダーの選択可能

## 4. インタラクション設計

| 操作 | 挙動 |
|------|------|
| タップ | タスクカード → 詳細/編集、チェックボックス → 完了 |
| ドラッグ&ドロップ | あとでリスト → タイムライン配置、優先順位の並び替え |
| スヌーズ | タスクの先送り（時間を後ろにずらす） |
| コピー | タスクの複製（v1.0.17〜） |
| Siri音声入力 | 「Hey Siri、Any Plannerでタスク追加」（v1.0.15〜） |

**注意**: タイムライン上のスワイプ時間移動は未対応。並び替えはボタン切替式（長押しドラッグではない）。

## 6. ADHD向け設計の分析

### 開発者の設計思想
開発者なつみかん氏自身がADHD当事者。「ADHDの脳のために設計された」と明言。

### 工夫の体系

| 対象課題 | 対応機能 | 設計意図 |
|---------|---------|---------|
| **タイムブラインドネス** | タイムライン + 空き時間表示 | 時間を「見える化」して時間感覚を補完 |
| **ワーキングメモリ不足** | 「あとでやる」リスト | 「覚えておく」プレッシャーを即解消 |
| **タスク開始困難** | 上から順にやる直線構造 | 複雑な優先度判断を排除 |
| **先延ばし** | スヌーズ機能 | 罪悪感なく先送り、でもタスクは消えない |
| **報酬系の弱さ** | 完了実績 + ストリーク | 小さな達成でもドーパミンを得られる |
| **過集中** | リマインダー通知 | 「次にやること」を外部から思い出させる |
| **認知負荷過多** | 機能を最小限、3択UI | 迷わせない、クラッター排除 |
| **習慣化困難** | 繰り返しタスク | 毎日の入力作業を自動化 |

## 7. 競合比較

### ポジショニングマップ

```
          シンプル
            ↑
            |
   Any Planner ● ← Structured ●
            |
  ADHD特化 ←|→ 一般向け
            |
    Tiimo ● |          ● Taskito
            |
            ↓
          高機能
```

### 詳細比較

| 項目 | Any Planner | Structured | Tiimo | Taskito | Routinery | BeforeSunset AI |
|------|------------|-----------|-------|---------|-----------|----------------|
| タイムライン | 縦型30分刻み | 縦型ブロック | 色分けビジュアル | フルスクリーン統合 | ステップ実行 | カレンダーブロック |
| 「あとで」リスト | ◎（コア機能） | △（インボックス） | △（ブレインダンプ→AI整理） | △（プロジェクト間移動） | × | △（AI自動リスケ） |
| D&D | あり | あり | あり | あり | あり | あり |
| ADHD特化度 | ★★★★ | ★★★★ | ★★★★★ | ★★ | ★★★★ | ★★★ |
| AI機能 | なし | Pro版 | コプランナー | なし | なし | コア機能 |
| 感覚カスタマイズ | テーマ切替のみ | なし | ◎（充実） | なし | 限定的 | なし |
| マルチプラットフォーム | ◎（全対応） | iOS/macOSのみ | iOS/Android/Web | iOS/Android/Web | iOS/Android | Web/iOS/Android |
| 無料版充実度 | 高 | 中 | 低 | 中 | 低 | 中 |
| 月額 | $5.99 | $6.49 | $12 | 不明 | $3.99〜 | $9.99 |
| 受賞歴 | — | — | App of the Year 2025 | — | App of the Day 2025 | — |

### Any Plannerの独自強み
1. **「あとで」を肯定するUX** — 競合にはない「先送りを正当化する」設計
2. **マルチプラットフォーム** — Structuredはios/macのみ、Tiimoはweb対応が弱い
3. **個人開発の速度** — 1ヶ月19バージョンは異例
4. **価格の手頃さ** — Tiimo($12/月)やBeforeSunset($9.99/月)より安価

### Any Plannerの弱み（競合対比）
1. **感覚カスタマイズ不足** — Tiimoが圧倒的に先行
2. **AI機能なし** — Tiimo/BeforeSunsetに劣る
3. **ルーティン実行ガイド** — Routineryの「1ステップ表示+タイマー」に相当する機能がない
4. **英語圏認知度** — Reddit/YouTube での言及ほぼゼロ

## 8. 我々のクローンへの提言

### 必須（Phase 1-2で対応すべき）

| 提言 | 根拠 |
|------|------|
| **空き時間の自動表示** | Any Plannerの核心機能。競合差別化の源泉 |
| **タスクの長さ=カード高さ** | Structured/Tiimoの成功要因。時間の物理的可視化 |
| **スワイプ/長押しドラッグの操作改善** | Any Planner本家への最多改善要望 |
| **完了時の「褒め」演出** | 小さな達成でも報酬感を得られるUX |

### 高優先（Phase 2-3で対応すべき）

| 提言 | 根拠 |
|------|------|
| **テーマ/カラーパレット切替** | Tiimoの感覚カスタマイズに学ぶ。ハイコントラスト/ソフトの選択 |
| **ルーティン実行モード** | Routineryの「1ステップずつ+タイマー」。朝/夜ルーティン向け |
| **先送りパターン分析** | 競合にない独自機能。何を何回先送りしたかの統計 |
| **未完了タスクの穏やかな表現** | 赤バツではなく、裁かないUIメッセージング |

### 中優先（Phase 3-4で検討）

| 提言 | 根拠 |
|------|------|
| **ブレインダンプ→AI整理** | Tiimoの差別化機能。将来のAI統合で対応 |
| **エネルギーレベルタグ** | Structuredの独自機能。認知リソース配分の支援 |
| **ホーム画面ウィジェット** | アプリ起動の摩擦を低減（ADHD向け必須） |
| **音声入力** | Any Planner本家がSiri対応。操作の即時性を高める |
