## 実装フェーズ

### Phase 1: MVP ✅
> タスクの追加・表示・ドラッグ配置ができる最小限

- [x] Flutter プロジェクト作成 + 依存関係セットアップ
- [x] データモデル（Task, UserSettings）+ Drift テーブル
- [x] パステルカラーテーマ基盤
- [x] タイムライン画面: 時間軸、タスクカード、現在時刻インジケーター
- [x] あとでやるリスト: ボトムシート、タスク一覧
- [x] タスク作成/編集: タイトル、タイプ（あとで/時間指定）、時刻設定
- [x] ドラッグ&ドロップ: あとでリスト → タイムライン
- [x] 起床/就寝時間設定
- [x] Windows / Android ビルド確認

### Phase 2: 日常利用機能
> 毎日使えるレベルにする

- [ ] 繰り返しタスク（Routine + RoutineGenerator）
- [ ] タスク完了/スヌーズ/スキップ操作
- [ ] ローカル通知（リマインダー、未完了リマインド）
- [ ] ピン留め
- [ ] パステルカラー選択（タスク別）
- [ ] カスタムアイコン + 自動サジェスト
- [ ] 設定画面UI

### Phase 3: 体験向上
> 使い込むほど便利になる機能

- [ ] OSカレンダー連携
- [ ] 長文メモ、画像添付
- [ ] 統計画面（完了数、ストリーク、時間帯別生産性）
- [ ] 日付移動（スワイプ/カレンダーピッカー）
- [ ] 空き時間インジケーター

### Phase 4: プラットフォーム最適化
> 各プラットフォームでのネイティブ体験

- [ ] ホーム画面ウィジェット
- [ ] 通知バッジ
- [ ] Web対応（Drift WASM）
- [ ] デスクトップ微調整
- [ ] パフォーマンス最適化

## 技術スタック

| 項目 | 選定 |
|------|------|
| フレームワーク | Flutter 3.41+ |
| 言語 | Dart 3.11+ |
| 状態管理 | Riverpod 2.x |
| ローカルDB | Drift (SQLite) |
| 不変モデル | freezed |
| ルーティング | go_router |
| 通知 | flutter_local_notifications（Phase 2） |
| カレンダー | device_calendar（Phase 3） |
| グラフ | fl_chart（Phase 3） |
